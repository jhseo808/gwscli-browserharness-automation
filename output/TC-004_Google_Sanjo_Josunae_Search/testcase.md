# TC-004: Google에서 "뿌리 깊은 산조" 검색 후 조순애 앨범 찾기

## 전제조건
- 브라우저: Chrome (browser-harness CDP 제어)
- 네트워크: 인터넷 연결 필요
- 로그인: 불필요
- Google 세션 쿠키 확립 필요 (CAPTCHA 방지, ERR-001 참조)

## 테스트 단계
| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | https://www.google.com 접속 | Google 홈페이지 로드 완료 |
| 2 | 검색창에 "뿌리 깊은 산조" 입력 후 검색 실행 | 검색 결과 페이지 로드 |
| 3 | 검색 결과에서 조순애 관련 앨범 링크 선택 | 앨범 또는 아티스트 페이지 로드 |

## 기대 결과
- ✅ Google 홈페이지 정상 접속
- ✅ "뿌리 깊은 산조" 검색 결과 노출
- ✅ 조순애 앨범 관련 페이지(멜론/벅스/유튜브 등) 진입

## 실패 조건
- ❌ Google CAPTCHA 차단 (/sorry/ URL)
- ❌ 검색 결과에 조순애 관련 항목 없음
- ❌ 앨범 링크 클릭 무반응

## 알려진 제약사항
- Google 세션 쿠키 없으면 CAPTCHA 발생 (ERR-001)
- 검색 결과는 동적으로 변할 수 있음
