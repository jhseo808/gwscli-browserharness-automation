# TC-002: 멜론 HOT100 2순위 노래 재생

## 전제조건
- Chrome 브라우저 정상 작동
- 멜론 웹 (www.melon.com) 접근 가능
- 멜론 계정 로그인 상태 (재생 기능 사용 시)

## 테스트 단계
| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | https://www.melon.com/index.htm 접속 | 멜론 홈페이지 로드 |
| 2 | 상단 네비게이션 "멜론차트" 클릭 | 멜론차트 페이지 이동 |
| 3 | 서브 탭 "HOT100" 클릭 | HOT100 차트 로드 |
| 4 | 2순위 노래의 "듣기" 버튼 클릭 | 음악 재생 시작 |

## 기대 결과
- ✅ 멜론 홈페이지 정상 로드
- ✅ HOT100 차트 페이지 접속 (URL: /chart/hot100/index.htm)
- ✅ 2순위 노래 듣기 버튼 클릭 반응
- ✅ 멜론 웹 플레이어 또는 플레이어 앱으로 재생

## 실패 조건
- ❌ 멜론 페이지 로드 실패
- ❌ HOT100 탭 없음
- ❌ 2순위 항목 없음
- ❌ 듣기 버튼 클릭 무반응

## 알려진 제약사항
- 멜론 웹 재생은 **멜론 데스크탑 플레이어 앱** 설치 필요
- 앱 미설치 환경(CI 포함)에서는 BLOCKED 처리
- 로그인 없이도 듣기 버튼 클릭 → 플레이어 실행 시도까지는 가능
