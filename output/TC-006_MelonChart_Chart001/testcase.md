# TC-006: 멜론차트 TOP100 순위1 앨범 정보 확인

## 전제조건
- Chrome 브라우저 실행 상태
- 인터넷 연결 정상
- 로그인 불필요

## 테스트 단계
| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | https://www.melon.com/index.htm 접속 | 멜론 홈 페이지 로드 완료 |
| 2 | 상단 GNB에서 "멜론차트" 클릭 | 멜론차트 TOP100 페이지로 이동 (URL: /chart/index.htm) |
| 3 | TOP100 차트 리스트 확인 | 100곡 순위 목록이 표시됨 |
| 4 | 순위 1위 앨범 이미지 클릭 | 해당 앨범 상세 정보 페이지로 이동 |
| 5 | 앨범 정보 확인 | 앨범명, 아티스트명 정상 출력 |

## 기대 결과
- ✅ 멜론차트 TOP100 페이지 정상 로드
- ✅ 1위 앨범 이미지 클릭 후 앨범 상세 페이지(albumId=13312398)로 이동
- ✅ 페이지 타이틀 "앨범 정보>멜론" 표시
- ✅ 앨범명("개화") 및 아티스트명("AKMU (악뮤)") 정상 출력

## 실패 조건
- ❌ 멜론차트 GNB 링크 미표시 또는 클릭 무반응
- ❌ TOP100 차트 목록 미로드
- ❌ 1위 앨범 이미지 없음 또는 클릭 후 앨범 상세 페이지 미이동
- ❌ 앨범 정보 페이지에서 앨범명 또는 아티스트명 미출력

## 알려진 제약사항
- 멜론차트 접속 시 TOP100이 기본 선택 상태 (별도 탭 클릭 불필요)
- 1위 곡은 실시간으로 변동될 수 있음 (albumId가 달라질 수 있음)
