# TC-001: YouTube BTS 검색 및 첫 번째 영상 재생

## 전제조건
- Chrome 브라우저 정상 작동
- 인터넷 연결 정상
- YouTube 접근 가능

## 테스트 단계
| 단계 | 액션 | 기대 결과 |
|------|------|-----------|
| 1 | YouTube 홈페이지 접속 | 홈페이지 로드 완료 |
| 2 | 상단 검색창 클릭 | 검색창 활성화 |
| 3 | "BTS" 입력 | 검색창에 텍스트 입력됨 |
| 4 | Enter 키 입력 | BTS 검색 결과 페이지 로드 |
| 5 | 첫 번째 비디오 클릭 | 비디오 재생 페이지 이동 |
| 6 | 비디오 로드 대기 | 플레이어 렌더링 완료 |
| 7 | 재생 여부 확인 | 비디오 재생 중 상태 |

## 기대 결과
- ✅ BTS 관련 영상 검색 결과 표시
- ✅ 첫 번째 영상 정상 재생
- ✅ 플레이어 UI 표시 (재생/일시정지, 진행률)
- ✅ 페이지 타이틀에 BTS 포함

## 실패 조건
- ❌ 검색창 활성화 안됨
- ❌ 검색 결과 없음
- ❌ 첫 번째 비디오 링크 없음
- ❌ 비디오 재생 실패
